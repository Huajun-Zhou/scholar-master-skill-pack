# Thinking Model Distillation Prompt

从 Paper Cards、Method Cards 和 synthesis 页面中提炼陈志远教授公开成果中体现出的科研思维模型。

## 思维模型不是口号

是可操作的推理链：

```text
现象 / 缺口 → 问题重构 → 关键假设 → 方法机制 → 实验验证 → 贡献表述
```

## 每个 Thinking Model 必须包含

1. **名称**：动作 + 对象格式（"从异常现象到机制建模"），不要标签式命名。
2. **描述**：一句话说明该模型如何帮助提出问题 / 设计方法 / 验证研究。
3. **证据来源**：≥ 2 篇论文，最好 ≥ 3 篇。
4. **典型推理链**：逐步可复现，不跳步。
5. **适用场景**。
6. **不适用场景**。
7. **迁移模板**：用户可直接套用的研究设计骨架。
8. **示例迁移**：跨陈志远教授原领域的新问题示例（演示可移植性）。

## 强制规则

- 单篇支持 → candidate，不入正式模型。
- B 类综合 vs C 类迁移必须显式标记（QG6）。
- 不冒充该学者表态偏好。
- 推理链中的每一步要么对应 evidence_id，要么标 inferred。

## 校验

`python scripts/validate_json.py thinking_model`。
