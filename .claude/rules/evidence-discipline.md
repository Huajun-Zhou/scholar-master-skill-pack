# Evidence Discipline Rule

## 核心规则

所有非平凡学术判断必须附 evidence_id。系统输出必须区分三类内容：

| 等级 | 名称 | 定义 | 输出要求 |
|------|------|------|----------|
| A | 直接证据 | 可在某篇论文中直接找到支持 | 必须附 evidence_id |
| B | 综合归纳 | 多篇论文共同体现的稳定模式 | 至少 2 篇论文支持 |
| C | 迁移推断 | 把学者方法论迁移到新问题 | 必须显式说明"这是推断" |

## 强制要求

1. A 类结论无 evidence_id 不得输出。
2. B 类结论少于 2 个 paper_id 降级为 C 或标记"证据不足"。
3. C 类结论必须显式标注"C 类迁移推断"。
4. 不确定内容必须写"不确定"或"证据不足"。
5. 不允许为了让答案完整而补充无证据结论。
